Converts a single pixel from C to FITS format.

"ctype" = PTYPE_FLOAT, PTYPE_INT, etc. "fitstype" = BPP_8_UNSIGNED, BPP_16_SIGNED, etc.

"cval": pointer to C value (input) "fitsval": pointer to FITS value (output)

Does byte-swapping, if necessary.

Converts a value described by a "PTYPE_" To a value described by a "BPP_"
