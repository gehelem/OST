bool StellarSolver::extract ( bool  calculateHFR = false,
QRect  frame = QRect() 
)

extract Performs Star Extraction on the image. This is performed synchronously and blocks the calling thread until the finished signal is emitted.

Parameters
calculateHFRIf true, it will also calculated Half-Flux Radius for each detected star. HFR calculations can be very CPU-intensive.
frameIf set, it will only extract stars within this rectangular region of the image.
Returns
A boolean that reports whether it was successful, true means success.
