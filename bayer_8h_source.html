232 /************************************************************************************************
233  * *
234  * Color conversion functions for cameras that can output raw Bayer pattern images (color *
235  * codings DC1394_COLOR_CODING_RAW8 and DC1394_COLOR_CODING_RAW16). *
236  * *
237  * Credits and sources: *
238  * - Nearest Neighbor : OpenCV library *
239  * - Bilinear : OpenCV library *
240  * - HQLinear : High-Quality Linear Interpolation For Demosaicing Of Bayer-Patterned *
241  * Color Images, by Henrique S. Malvar, Li-wei He, and Ross Cutler, *
242  * in Proceedings of the ICASSP'04 Conference. *
243  * - Edge Sense II : Laroche, Claude A. "Apparatus and method for adaptively interpolating *
244  * a full color image utilizing chrominance gradients" *
245  * U.S. Patent 5,373,322. Based on the code found on the website *
246  * http://www-ise.stanford.edu/~tingchen/ Converted to C and adapted to *
247  * all four elementary patterns. *
248  * - Downsample : "Known to the Ancients" *
249  * - Simple : Implemented from the information found in the manual of Allied Vision *
250  * Technologies (AVT) cameras. *
251  * - VNG : Variable Number of Gradients, a method described in *
252  * http://www-ise.stanford.edu/~tingchen/algodep/vargra.html *
253  * Sources import from DCRAW by Frederic Devernay. DCRAW is a RAW *
254  * converter program by Dave Coffin. URL: *
255  * https://dechifro.org/dcraw/ *
256  * - AHD : Adaptive Homogeneity-Directed Demosaicing Algorithm, by K. Hirakawa *
257  * and T.W. Parks, IEEE Transactions on Image Processing, Vol. 14, Nr. 3, *
258  * March 2005, pp. 360 - 369. *
259  * *
260  ************************************************************************************************/
