Detailed Description

Template classes for typed elements with numeric/non-numeric specializations.

Author
OST Development Team
Version
1.0

This file defines the template hierarchy that implements typed elements in OST. The template system splits into two branches:

  • ElementTemplateNumeric<T>: For numeric types (int, float) with min/max/step/slider support
  • ElementTemplateNotNumeric<T>: For non-numeric types (string, bool, date, time, etc.)

Also defines ValueTemplate<T> for grid value storage.
