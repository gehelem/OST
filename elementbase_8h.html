Detailed Description

Base classes for OST typed elements and their value containers.

Author
OST Development Team
Version
1.0

This file defines ElementBase (base for all typed elements like int, float, string) and ValueBase (base for value containers used in grid lines). Elements hold typed values and metadata, while Values are lightweight copies for grid storage.
