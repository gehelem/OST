110 // Notes for the function below:
111 // Return the full path to index files to use when solving.
112 // The input is a list of directory names, and index and healpix constraints.
113 // If indexToUse and healpixToUse are -1, then return all the .fit or .fits
114 // files in the directories. If indexToUse >= 0, then constrain the list to
115 // just those of the correct index. If healpixToUse is also >= 0 then
116 // further constrain the list to correct healpix. It is assumed the index
117 // filename format is index-$INDEX.fit or .fits, or index-$INDEX-$HH.fit or
118 // .fits where $HH is a 2-character healpix number, e.g. index-4205-03.fits.
119
