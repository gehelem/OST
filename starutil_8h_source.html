215/*
216 Computes the 2D coordinates (x,y) (in units of a celestial sphere of
217 radius 1) that star s would have in a TANGENTIAL PROJECTION defined
218 by (centred at) star r. s and r are both given in xyz coordinates,
219 the parameters are pointers to arrays of size 3.
220
221 If "tangent" is true, the projection is the WCS TAN projection; if
222 not, it is the WCS SIN projection.
223
224 The resulting x,y coordinates are intermediate world coordinates in
225 degrees. The "x" direction should be positive for increasing RA, "y"
226 is increasing Dec.
227
228 WARNING -- this code assumes s and r are UNIT vectors (ie normalized
229 to have length 1).
230*/
