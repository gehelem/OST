Fit polynomial using GSL multi-parameter fitting.

Performs least-squares polynomial fit: y = c[0] + c[1]*x + c[2]*x² + ... Using GSL's robust linear algebra routines with covariance matrix calculation.

Algorithm:

  1. Build design matrix X where X[i,j] = x[i]^j
  2. Setup vector y with data points
  3. Solve linear system: X*c = y (least squares)
  4. Return coefficient vector c and chi-square error
