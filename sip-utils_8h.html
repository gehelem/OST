)

sip->a_order, sip->b_order, sip->a, and sip->b must be set.

sip->ap_order and sip->bp_order must be set.

Computes sip->ap and sip->bp by inverting the forward polynomial evaluated on a regular grid of points, NX x NY, between xlo,xhi and ylo,yhi.

If NX,NY are 0, a reasonable default will be chosen.

If xlo=xhi=0 or ylo=yhi=0, the bounds of the image (from sip->wcstan.imagew/h) will be used.

Returns the bounds of the image in RA,Dec space, approximately, by walking the boundary of the image in steps of size "stepsize".

Return values satisfy: -90 <= decmin <= decmax <= 90 ramin <= ramax ramin may be < 0, or ramax > 360, if the image straddles RA=0.
