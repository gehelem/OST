Detailed Description

Image data container for grid storage.

ValueImg is the grid value copy for ElementImg. It stores ImgData structures and provides methods to synchronize with its parent ElementImg.

Useful for storing image sequences in grids (e.g., focus run frames, guiding log with frame references).

Usage (internal to PropertyMulti):
// PropertyMulti::push() creates values from elements
auto valueImg = new ValueImg(previewElement);
valueImg->updateValue(); // Copy element->value() to valueImg->value
gridLine["preview"] = valueImg;
// PropertyMulti::fetchLine() restores elements from values
valueImg->updateElement(true); // Copy valueImg->value to element->value()
